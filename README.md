- Compétences avec barres (rouge/orange/vert)

- Thème vert, responsive, prêt pour GitHub Pages

## Déploiement
1. Crée un dépôt GitHub et push tout le contenu de ce dossier à la racine.
2. Dans **Settings → Pages**, choisis la branche `main` et le dossier `/ (root)`.
3. L’URL de ton site sera disponible en quelques minutes.
